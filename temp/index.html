Первинна гiдроiзоляцiя пожежного
резервуару

с. Адампiль, Хмельницька обл.

Компанiя "БІГ Київ" забезпечила первинною гідроiзоляцiєю пожежний резервуар маслоекстракцiйного заводу в с. Адампіль, Хмельницької областi. Згiдно з вимогами проектної документацiї, бетон резервуару повинен був мати показник водонепроникностi не менш W12. Для забезпечення цiєї вимоги, замовником було прийнято рiшення застосувати гiдроiзоляцiйну добавку для бетону Пенетрон Адмiкс i шнур Пенебар для гiдроiзоляцiї холодних швiв i проходiв комунiкацій.

Гiдроiзоляцiя пiдземних галерей
на Носiвському ХПП

м. Носiвка, Чернiгiвська область

Наша компанiя виконала комплекс робiт з гiдроiзоляцiї норiйного приямку i пiдземних галерей при будiвництвi нової черги елеватора Носiвського ХПП, в смт. Носiвка, Чернiгiвська обл. Для забезпечення герметичностi цих пiдземних споруд застосовується система проникаючої гiдроiзоляцiї Пенетрон - проникаючий матерiал для захисту бетону вiд капiлярних протiкань i шовний матерiал Пенекрит для герметизацiї холодних швiв, отворiв вiд опалубки i примикань.

Гiдроiзоляцiя норiйних приямкiв насiнної фабрики "Євросiд"

с. Дулицьке, Київська обл.

Насіннєва фабрика "Евросiд" займається переробкою, протруєнням та упаковкою посiвного матерiалу.
